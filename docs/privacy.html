Privacy Policy

Claude Code Dashboard — Last updated April 14, 2026

Overview

Claude Code Dashboard is a local desktop application that reads configuration files from your computer to display them in a visual interface. Your privacy is straightforward: we collect nothing.

Data Collection

Claude Code Dashboard does not collect, transmit, store, or share any personal data. Specifically:

  • No analytics or telemetry
  • No network requests to external servers
  • No user accounts or authentication
  • No cookies, tracking pixels, or advertising
  • No crash reporting sent externally

Local Data Access

The app reads files on your local filesystem to display your Claude Code configuration:

  • ~/.claude/ — global Claude Code settings, skills, agents, commands, memory, and plans
  • ~/Documents/GitHub/ — repository-level Claude Code configurations

All data stays on your machine. Nothing is sent over the network. The app writes to your filesystem only when you explicitly toggle a permission setting, which modifies ~/.claude/settings.json.

Third-Party Services

Claude Code Dashboard does not integrate with or send data to any third-party services. The app loads one external resource: the Inter font from Google Fonts, which is subject to Google's Privacy Policy.

Children's Privacy

This app is not directed at children under 13 and does not knowingly collect information from children.

Changes to This Policy

If this policy changes, the updated version will be posted at this URL. Since we collect no data, meaningful changes are unlikely.

Contact

Questions about this policy can be directed to the project's GitHub Issues.
